# コマンドライン タイピングゲーム (Java)

大学の授業課題にて制作した、ターミナル上で動作するタイピング練習ゲームです。
ランダムで割り振られた他の受講生が作成した企画書をもとに、それを再現・実装した作品となります。

## 特徴・工夫した点
* **他者の企画書をもとにした仕様実装**
  他人が考案したゲームのルールや画面遷移の企画書を解釈し、コードに落とし込んで完成させました。
* **設計（MVCパターン）**
  表示（View）、制御（Controller）、データ処理（Model）を明確に分離し、コードの役割分担と保守性を意識して設計・実装しました。
* **スコア・コンボ・タイマー機能**
  HP（体力）システム、連続正解（コンボ）のカウント、回答制限時間タイマーなどのゲーム要素を実装しています。

## 動作環境
* Linux (Bash環境)
* Java 11 以上

## 実行方法

1. シェルスクリプトへの実行権限付与
chmod +x read_key.sh
2. コンパイル
javac *.java
3. 実行
./read_key.sh | java Model
